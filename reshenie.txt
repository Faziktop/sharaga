**1. Методологии User Story Mapping и Impact Mapping (кратко)**

**User Story Mapping** — метод формирования бэклога продукта через визуализацию шагов пользователя («activities» → «tasks» → «stories»). Позволяет увидеть целостную картину взаимодействия с системой, выделить MVP и зоны для доработки.

**Impact Mapping** — стратегический метод планирования, связывающий бизнес-цель («Why») → действующих лиц («Who») → способы влияния («How») → функции («What»). Помогает увязать функционал с измеримыми результатами для бизнеса.

**5. Пояснительная записка (обоснование приоритетов)**

**Приоритет №1 — «Умное напоминание: 3 касания до визита»**

Прямое влияние на ключевую проблему клиники — 30% неявок. Быстрое внедрение (32 ч), высокий ROI. Без этого шага дальнейшая аналитика теряет смысл — сначала надо снизить потери, потом их измерять.

**Приоритет №2 — «Автообзвон + Smart-лист неподтвердивших»**

Снижает ручной труд администраторов, которые перегружены. Позволяет переиспользовать освободившиеся слоты. Но требует базовой функции напоминаний (№1) как фундамента.

**Приоритет №3 — «Аналитика неявок»**

Важна для стратегии, но не даёт мгновенного эффекта. Её ценность раскрывается после автоматизации напоминаний — тогда дашборд покажет остаточный % неявок и поможет точечно донастроить политику.

**Итоговая рекомендация:** внедрять Feature 1 → Feature 2 параллельно (бэкенд общий), Feature 3 — в следующем спринте.

**6. Глоссарий**

| Термин | Определение |

| Автообзвон (робот-обзвон) | Автоматическая телефонная рассылка с голосовым сценарием без участия оператора |
| Smart-лист | Динамический список пациентов, сформированный по правилам (например, «не подтвердили запись за 12 часов») |
| ROI | Return on Investment — окупаемость вложений |

### Шаг 3. Оформите документ в Word

1. Выделите заголовки и сделайте их **жирным шрифтом**
2. Для таблиц используйте **Вставка → Таблица** (скопированные таблицы встанут как текст — преобразуйте их в таблицы через меню Word)
3. Установите поля: **2 см со всех сторон**
4. Шрифт: **Times New Roman, 12 кегль**, межстрочный интервал **1,5**
5. Вверху справа укажите свои ФИО и группу
