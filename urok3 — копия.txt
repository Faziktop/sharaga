Отлично, теперь понятно — вы работаете в **POV-Ray** и вам нужно зеркально отобразить сцену освещения и светильник с одной стены коридора на другую.

Судя по коду, у вас есть:
- Правая стена (x ≈ 14..15) с источником света `light_source` и "плафоном" (box с фильтром).
- Левая стена (x ≈ -5..5) пока без симметричного освещения.

## Как отзеркалить светильник на другую стену

### 1. Определяем центр коридора
По вашим объектам: правая стена в районе **x = 14..15**, левая — в районе **x = -5..5**.  
Центр зеркалирования примерно **x ≈ 4.5..5**.

### 2. Добавляем зеркальный светильник на левую стену

Вставьте этот код **после** правого светильника:

```pov
// =============================================
// ЗЕРКАЛЬНЫЙ СВЕТИЛЬНИК НА ЛЕВОЙ СТЕНЕ
// =============================================

// Источник света (отзеркаленный)
light_source {
    <-5, 8, -320>    // Зеркальное положение: (4.5 - (14 - 4.5)) = -5
    
    color rgb <2.5, 2.3, 2.0>
    
    fade_distance 20
    fade_power 20.0
}

// Плафон / корпус светильника (зеркально)
box { 
    <-6, 3-5, -321>,           
    <-5, 13+5, -319>             
    texture { pigment 
            { color White filter 0.8}
            finish {
                diffuse 1 
            }            
    }
    no_shadow
}
```

### 3. (Опционально) Добавить визуальный эффект свечения

Чтобы левый светильник выглядел как включённый:

```pov
// Сфера свечения вокруг левого светильника
sphere {
    <-5.5, 8, -320>, 2
    texture { pigment { color rgbt <1, 0.9, 0.7, 0.7> } }
    finish { diffuse 0 emission 0.6 }
    no_shadow
}
```

### 4. Полный готовый блок для вставки

```pov
// ========= ЛЕВЫЙ СВЕТИЛЬНИК (ЗЕРКАЛО ПРАВОГО) =========
light_source {
    <-5, 8, -320>
    color rgb <2.5, 2.3, 2.0>
    fade_distance 20
    fade_power 20.0
}

box { 
    <-6, -2, -321>,           
    <-5, 18, -319>             
    texture { pigment { color White filter 0.8 }
             finish { diffuse 1 }
    }
    no_shadow
}

// Эффект свечения (мягкий свет на стене)
sphere {
    <-5.5, 8, -320>, 1.5
    pigment { rgbt <1, 0.95, 0.8, 0.6> }
    finish { emission 0.5 }
    no_shadow
}
```

### 5. Результат
- Правый светильник: x ≈ 14..15
- Левый светильник: x ≈ -6..-5
- Оба светят с одинаковой яркостью и затуханием
- Визуально симметрично

### 6. Если нужно привязать к существующему выключателю (логика)

В POV-Ray выключатели не моделируются, но если нужно показать, что они включаются синхронно — просто добавьте оба `light_source` с одинаковыми параметрами.
